Editing

Markdown Syntax:

  • [link text](SomePage): Internal link to SomePage
  • # Section, ## Subsection: Section headers
  • 1. Item, <space>1. Second-level item: Enumerated list
  • * Item, <space>* Second-level item: Itemized list
  • [Some URL](http://someurl.com): External links
  • [![image description](http://image.url)](http://image.url): Embedded images
  • *Emphasised text*, **bold text**, _italic text_ → Emphasized text, bold text, italic text
For more markup styles, see the document.

 

Textile Syntax: Besides normal HTML code (e.g. <b>Bold</b>), the following markup is available as well:

  • "link text":SomePage: Internal link to SomePage
  • h1. Section, h2. Subsection: Section headers
  • # Item, ## Second-level item: Enumerated list
  • * Item, ** Second-level item: Itemized list
  • "Some URL":http://someurl.com: External links
  • !/path/to/image.jpg!: Embedded images
  • _Emphasised text_, *strong text*, ??citations??, @code@, +Inserted text+, -Removed text- → Emphasized text, strong text, citation, inserted text, removed text
  • H~2~O, A^2^ → H2O, A2
  • Abbr(Abbreviation) → Abbr
  • |Cell 1|Cell 2|: Tables
  • %{color:red}Red% text → Red text
For more markup styles, see the Tutorial, Textile reference, and comprehensive documents on txstyle.org .
